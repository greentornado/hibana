## 4. Real-World Feature Demonstration

### A. Router Performance

**Test:** 1000+ routes with O(1) dispatch

```elixir
# CompiledRouter generates at build time:
def match("GET", ["static", n]), do: static_route(n)
def match("GET", ["user", id]), do: user_route(id)
# ... 1000+ clauses
```

**Result:** Constant-time dispatch regardless of route count

### B. File Streaming

**Test:** Download 10MB file via sendfile(2)

```bash
curl -O http://localhost:4008/download/test_large.bin
# Downloaded: 10MB in <1s
# No memory overhead - kernel-space transfer
```

**Result:** Zero-copy file transfer works correctly

### C. SSE Streaming

**Test:** Server-Sent Events with auto-reconnect

```bash
curl -N http://localhost:4008/events
# Output:
retry: 3000
event: connected
data: {"message":"SSE connection established"}
event: update
data: {"counter":1}
```

**Result:** Real-time streaming functional

### D. Range Requests

**Test:** Partial content download

```bash
curl -H "Range: bytes=0-1023" http://localhost:4008/download/test_small.bin
# HTTP/1.1 206 Partial Content
# Content-Range: bytes 0-1023/1048576
```

**Result:** Video seeking/resumable downloads work
