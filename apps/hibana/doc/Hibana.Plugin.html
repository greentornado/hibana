    Hibana.Plugin behaviour (Hibana v0.1.0)

    Copy Markdown View Source

    Behaviour for Hibana plugins.

    Example

    defmodule MyApp.AuthPlugin do
  @behaviour Hibana.Plugin

  def init(opts) do
    # Initialize plugin with options
    opts
  end

  def call(conn, _opts) do
    # Process the request
    conn
  end

  def before_send(conn, _opts) do
    # Called before response is sent
    conn
  end
end

    Summary

    Callbacks

    before_send(conn, opts)

    Called just before the response is sent. Use this for cleanup or adding response headers.

    call(conn, opts)

    Called for each request. This is the main entry point for the plugin. Must return the connection, possibly with modifications.

    init(opts)

    Called when the plugin is initialized. Use this to set up state and validate options.

    start_link(opts)

    Optional. Called when the plugin is started.
