    Hibana.Job (Hibana v0.1.0)

    Copy Markdown View Source

    Background job queue module using OTP.

    Usage

    defmodule MyJob do
  use Hibana.Job

  def perform(data) do
    IO.inspect(data, label: "Processing job")
  end
end

# Enqueue a job
MyJob.enqueue(%{user_id: 123, action: "send_email"})
